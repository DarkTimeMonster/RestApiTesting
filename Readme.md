# 🚀 Task Tracker REST API

Pet-проект бэкенд-сервиса для управления задачами. Реализован полноценный асинхронный цикл CRUD с подключением к реальной реляционной базе данных через ORM.

## 🛠 Технологический стек
* **Язык:** C# / .NET
* **Фреймворк:** ASP.NET Core Web API
* **База данных:** PostgreSQL
* **ORM:** Entity Framework Core (подход Code-First)

## ⚙️ Ключевые архитектурные решения
* Реализована строгая маршрутизация REST API (`[HttpGet]`, `[HttpPost]`, `[HttpPut]`, `[HttpDelete]`).
* Использована асинхронная обработка запросов (`async / Task / await`) для предотвращения блокировки потоков.
* Настроено внедрение зависимостей (Dependency Injection) для `AppDbContext`.
* Реализована защита от конфликтов временных зон PostgreSQL через принудительную конвертацию `DateTime` в формат UTC.

## 📡 API Эндпоинты

| HTTP Метод | Маршрут | Описание |
| :--- | :--- | :--- |
| `GET` | `/api/tasks` | Получить список всех задач |
| `POST` | `/api/tasks` | Создать новую задачу |
| `PUT` | `/api/tasks/{id}` | Обновить существующую задачу |
| `DELETE` | `/api/tasks/{id}` | Удалить задачу |
